机器学习基石上（第1周）学习笔记

发表于 2018-04-01

1.课程介绍

林轩田老师介绍了“机器学习基石（上）”的课程安排。

2.什么是机器学习

学习：获取技能

观察 ->学习 ->技能

机器学习：获取技能

数据 ->机器学习(ML) ->技能

而

技能 <=> 提高某种表现能力

机器学习也可以是

数据 ->机器学习(ML) ->提高某种表现能力

例如机器学习在金融领域的应用：

股票数据 ->机器学习(ML) ->提高收益

3.机器学习应用

日常需要：吃、穿、住、行
比如电影推介系统：
数据：用户给电影评分数据
能力：预测一个用户给某部未评分的电影打分

用户对电影的喜好特征 <----进行匹配（内积等）----> 电影的自带特征

这样就能预测用户对某部电影的评分，判断是否给该用户推荐此电影

4.机器学习组成部分

例如，申请银行信用卡，下图是用户填的信息：

而银行可以根据这些信息判断是否给该用户发信用卡。
上面的例子就有机器学习的基本组成：

输入：x ∈ X （用户信息）
输出：y ∈ Y （银行是否发卡）
目标函数：f: X -> Y （理想信用卡审核模型）

数据 <=> 训练集： D = {(x1, y1), (x2, y2), ... ,(xn, yn)} （银行信用卡历史申请人数据）
假设（hypothesis）<=> 能力：g:X -> Y（通过数据学习的信用卡审核模型）

通常情况下 g 与 f 越接近越好。

未知目标函数：f: X -> Y
|
训练集D -> 学习算法A -> hypothesis 函数 g ≈ f
